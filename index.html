Conceptos Fundamentales

Es la capacidad para obtener la ubicación geográfica real de un objeto o persona mediante datos digitales.
Fórmula base: Coordenadas (Latitud, Longitud).

  • GPS: Satelital (alta precisión).
  • IP: Basado en la red.
  • Wi-Fi: Triangulación de puntos de acceso.
  • Celdas Móviles: Antenas de telefonía.

  1. Google Maps (Navegación)
  2. Uber (Transporte)
  3. Tinder (Social)
  4. Instagram (Geo-etiquetado)
  5. Rappi (Logística)

Es una Application Programming Interface. Es el puente que permite que nuestro sitio web 'hable' con el servidor de mapas (como Leaflet o Google Maps).
